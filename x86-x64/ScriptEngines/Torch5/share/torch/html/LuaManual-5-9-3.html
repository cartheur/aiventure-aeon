debug.gethook ([thread])

Torch Manual  >   Lua 5.1 Reference Manual  >   Standard Libraries  >   The Debug Library  >   debug.gethook ([thread])

Returns the current hook settings of the thread, as three values: the current hook function, the current hook mask, and the current hook count (as set by the debug.sethook ([thread,] hook, mask [, count]) function).
