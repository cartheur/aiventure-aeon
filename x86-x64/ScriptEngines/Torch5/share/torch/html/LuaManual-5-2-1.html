coroutine.create (f)

Torch Manual  >   Lua 5.1 Reference Manual  >   Standard Libraries  >   Coroutine Manipulation  >   coroutine.create (f)

Creates a new coroutine, with body f. f must be a Lua function. Returns this new coroutine, an object with type "thread".
