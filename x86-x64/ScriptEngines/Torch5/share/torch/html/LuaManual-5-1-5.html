_G

Torch Manual  >   Lua 5.1 Reference Manual  >   Standard Libraries  >   Basic Functions  >   _G

A global variable (not a function) that holds the global environment (that is, _G._G = _G). Lua itself does not use this variable; changing its value does not affect any environment, nor vice-versa. (Use setfenv (f, table) to change environments.)
