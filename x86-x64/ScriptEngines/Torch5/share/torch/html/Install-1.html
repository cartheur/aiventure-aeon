Installing from binaries

Torch Manual  >   Torch Installation Manual  >   Installing from binaries

We provide binary installers for several common platform. It allows you to try out quickly Torch. For extensive usage (or if you have another platform!), we strongly encourage installing from the sources.

For all platforms we provide several executables:

lua
Basic command line Lua. Does not support graphics. See lua -h for help.
qlua
Command line or IDE Lua. Handles graphics and IDE. See qlua -h for help. Run qlua -ide if you like IDE environments.

We also provide additional specific platform executables:

MacQLua
IDE application available on MacOS X (corresponds to qlua -ide).
winqlua
IDE application available on Windows (corresponds to qlua -ide).
