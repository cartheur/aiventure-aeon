addT4dotT2(value, t4, t2)

Torch Manual  >   Torch Package Reference Manual  >   Tensor  >   Basic operations  >   addT4dotT2(value, t4, t2)

Performs a tensor multiplication between t4 (4D tensor) and t2 (2D tensor), multiply the resulting matrix by the scalar value and add the result to self (2D tensor). In other words, 

self = self + value * t4*t2

If t4 is a n x m x p x q tensor, t2 a p x q matrix, self must be a n x m matrix.
