torch.TYPEStorage([size])

Torch Manual  >   Torch Package Reference Manual  >   Storage and MapStorage  >   torch.TYPEStorage([size])

Returns a new Storage of type TYPE. Valid TYPE are Char, Short, Int, Long, Float, and Double. If size is given, resize the Storage accordingly, else create an empty Storage.

Example: 

-- Creates a Storage of 10 double:
x = torch.DoubleStorage(10)

The data in the Storage is uninitialized.
