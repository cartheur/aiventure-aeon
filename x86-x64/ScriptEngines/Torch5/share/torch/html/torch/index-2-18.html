Torch Manual  >   Torch Package Reference Manual  >   File  >   quiet()

If this mode is chosen instead of pedantic(), no Lua error will be generated in case of read/write error. Instead, a flag will be raised, readable through hasError(). This flag can be cleared with clearError()

Checking if a file is quiet can be performed using isQuiet().
