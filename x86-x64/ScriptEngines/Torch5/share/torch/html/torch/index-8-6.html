[boolean] torch.isequal(object1, object2)

Torch Manual  >   Torch Package Reference Manual  >   Torch utility functions  >   [boolean] torch.isequal(object1, object2)

If the two objects given as arguments are Lua tables (or Torch objects), then returns true if and only if the tables (or Torch objects) have the same address in memory. Returns false in any other cases.

A Torch class is a class created with torch.class() or torch.newmetatable().
