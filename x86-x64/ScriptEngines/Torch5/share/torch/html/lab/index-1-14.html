lab.triu(x)

Torch Manual  >   Lab Package Reference Manual  >   Construction or extraction functions  >   lab.triu(x)

y=lab.triu(x) returns the upper triangular part of x, the other elements of y are set to 0.

lab.triu(x,k) returns the elements on and above the k-th diagonal of x as non-zero. k=0 is the main diagonal, k>0 is above the main diagonal and k<0 is below the main diagonal.
