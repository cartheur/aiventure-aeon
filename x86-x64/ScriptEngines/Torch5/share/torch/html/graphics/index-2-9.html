surface:write_to_png(filename)

Torch Manual  >   Portable Graphic Layer Reference Manual  >   Drawing Surfaces  >   surface:write_to_png(filename)

Expression surface:write_to_png(filename) writes the content of the window or image surface surface into the PNG file filename. This function is not available for PDF and PS surfaces.
