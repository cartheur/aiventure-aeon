graphics.newps(w,h,filename)

Torch Manual  >   Portable Graphic Layer Reference Manual  >   Functions  >   graphics.newps(w,h,filename)

Expression graphics.newps(w,h,filename) returns drawing surface descriptor for a PostScript file filename. Arguments w and h specify the size of the drawing area.
