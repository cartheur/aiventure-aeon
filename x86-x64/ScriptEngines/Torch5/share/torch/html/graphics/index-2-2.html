surface:theEnd()

Torch Manual  >   Portable Graphic Layer Reference Manual  >   Drawing Surfaces  >   surface:theEnd()

Expression surface:theEnd() terminates a segment of Lua code that draws on the surface. Calls to surface:begin and surface:theEnd must be balanced. Drawings only appear on the surface after the last call to surface:theEnd. However you can use surface:refresh() at any time to cause an immediate display of the current drawings

  surface:begin()
surface:refresh()
