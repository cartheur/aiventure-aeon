file:write (...)

Torch Manual  >   Lua 5.1 Reference Manual  >   Standard Libraries  >   Input and Output Facilities  >   file:write (...)

Writes the value of each of its arguments to the file. The arguments must be strings or numbers. To write other values, use tostring (e) or string.format (formatstring, ...) before write.
