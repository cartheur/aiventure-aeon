qtide.doeditor(qluaeditor)

Torch Manual  >   QtIde Package Reference Manual  >   Functions  >   qtide.doeditor(qluaeditor)

This function executes the chunk of Lua code corresponding to the text present in the editor qluaeditor. This is the function called by the IDE when one selects the menu entry "_Load Lua File_" from an editor whose contents is not associated with a file or has been modified (otherwise one can use the standard Lua function dofile.)
