qtuiloader.load(filename,[parent])

Torch Manual  >   QtUiLoader Package Reference Manual  >   Functions  >   qtuiloader.load(filename,[parent])

Expression qtuiloader.load(filename,parent) dynamically constructs and returns the widget described by the Qt Designer file filename.

When argument parent is nil or not specified, the new widget is owned by the Lua interpreter and is automatically destroyed when the garbage collector determines that it is no longer referenced.
