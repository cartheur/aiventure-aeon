set(variable,value)

Torch Manual  >   Plotting Package Reference Manual  >   Window  >   Drawing primitives  >   set(variable,value)

Sets the value of a variable such as penColor or fontSize for an object. Typical example: 

	w=gfx.Window();
	p=w:plot({1,2,3,4})
	w:set('fontSize',20)
If an object such the window as in this example has children, then all the variables for its children are changed too. So this would change the font size for all items in the plot. If you want to change the font size for only a single object, you should apply the set command to its handle instead: 
	w=gfx.Window();
	p=w:plot({1,2,3,4})
	p2=w:legend('hi');
	p2:set('fontSize',40)
Calling h:set() with no arguments for a given object h returns the variables currently set in that object, and their assignments.
