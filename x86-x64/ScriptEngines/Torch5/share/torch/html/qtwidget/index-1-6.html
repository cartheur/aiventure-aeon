Painting functions.

Torch Manual  >   QtWidget Package Reference Manual  >   High Level Interface  >   Painting functions.

All the painting functions defined by class qt.QtLuaPainter can be called directly using the Lua method syntax on the table returned by the functions qtwidget.newwindow, qtwidget.newimage, qtwidget.newps, qtwidget.newpdf, and qtwidget.newprint.

These calls are simply forwarded to the underlying qt.QtLuaPainter object.
