painter:printer()

Torch Manual  >   QtWidget Package Reference Manual  >   QtLua Classes  >   qt.QtLuaPainter  >   QtLuaPainter Functions  >   painter:printer()

Expression painter:printer() returns the Qt object of class qt.QtLuaPrinter underlying the painter if it is associated with a printer object. Otherwise returns nil.
