painter:device()

Torch Manual  >   QtWidget Package Reference Manual  >   QtLua Classes  >   qt.QtLuaPainter  >   QtLuaPainter Functions  >   painter:device()

Expression painter:device() returns a pointer to the Qt paint device underlying the painter. Since the Qt class QPaintDevice is neither a Qt variant not a Qt object, this pointer is represented by a Qt variant of type void*. This is not very useful in Lua.
