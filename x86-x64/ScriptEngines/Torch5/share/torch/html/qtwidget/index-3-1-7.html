[QtLuaListener signal] sigMouseRelease(int,int,QByteArray,QByteArray,QByteArray)

Torch Manual  >   QtWidget Package Reference Manual  >   QtLua Classes  >   qt.QtLuaListener  >   [QtLuaListener signal] sigMouseRelease(int,int,QByteArray,QByteArray,QByteArray)

Signal sigMouseRelease is emitted whenever the underlying widget receives a QMouseEvent of type QEvent::MouseButtonRelease because a mouse button was released. The first two arguments are the pixel coordinates of the mouse pointer when the button was released The third argument is a string containing the [[http://doc.trolltech.com/main-snapshot/qt.html#MouseButton-enum][button name]. The fourth argument is a string containing the [[http://doc.trolltech.com/main-snapshot/qt.html#KeyboardModifier-enum][keyboard modifier names] separated by a vertical bar. The fifth argument is a string containing the names of all depressed buttons separated by a vertical bar.
