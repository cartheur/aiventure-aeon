qtwidget.newwindow(w,h,[title])

Torch Manual  >   QtWidget Package Reference Manual  >   High Level Interface  >   qtwidget.newwindow(w,h,[title])

Expression qtwidget.newwindow(w,h,title) creates a toplevel widget of class qt.QWidget and a qt.QtLuaPainter object operating on that widget. Numerical arguments w and h specify the size of the widget. String argument title specifies a window title. The expression returns a Lua table that serves as a drawing port descriptor.
