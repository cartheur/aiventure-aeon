printer.pageSize

Torch Manual  >   QtWidget Package Reference Manual  >   QtLua Classes  >   qt.QtLuaPrinter  >   printer.pageSize

String property printer.pageSize contains the name of the paper format. The recognized names are "A4", "B5", "Letter", "Legal", "Executive", "A0", "A1", "A2", "A3", "A5", "A6", "A7", "A8", "A9", "B0", "B1", "B10", "B2", "B3", "B4", "B6", "B7", "B8", "B9", "C5E", "Comm10E", "DLE", "Folio", "Ledger", and "Tabloid".

This property can also contain the value "Custom" to indicate that the paper format has been determined by other means, for instance using property printer.paperSize with recent versions of Qt.
