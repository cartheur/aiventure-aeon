Program qlua

Torch Manual  >   QtLua and Qt Package Reference Manual  >   Program qlua

Program qlua is a compatible replacement for the standard interpreter program lua with several new capabilities:

  • New packages provides a rich interface with the Qt4 library: qt, qtwidget, qttorch, qtuiloader.

  • The Lua interpreter runs in a dedicated thread. The main thread remains in charge of graphical interface and user interaction. Therefore all graphical interfaces remain active while the interpreter is running.

  • Console output can be captured and emitted as Qt signals. Therefore one can program a complete integrated development environment including a graphical console.

  • When using qlua, packages graphics and gfx produce drawings using Qt instead of using the Cairo library.
