qt.qEngine.lastErrorLocation

Torch Manual  >   QtLua and Qt Package Reference Manual  >   Package qt  >   Classes  >   qt.qEngine  >   qt.qEngine.lastErrorLocation

Read-only property qt.qEngine.lastErrorLocation contains a QStringList whose elements describe the program locations extracted from the stack when the last error message was recorded. When a location corresponds to a line in a file, the string has the format "@filename:linenumber".
