Named Children

Torch Manual  >   QtLua and Qt Package Reference Manual  >   Package qt  >   Qt Objects  >   Named Children

All Qt objects can be given a name represented by property objectName. The named children of a Qt object can be accessed by name using the indexation syntax. However, when the name of a child conflicts with a property or an exposed method, the property or the exposed method has precedence.
