qt.disconnect(qobject, signature, function)

Torch Manual  >   QtLua and Qt Package Reference Manual  >   Package qt  >   Functions  >   qt.disconnect(qobject, signature, function)

Expression qt.disconnect(qobject, signature, func) disconnects the signal signature from Qt object qobject from the Lua function func. Arguments signature or function may be nil. All connections matching the non-nil arguments will then be disconnected.

  qt.connect(qobject1, signature1, qobject2, signature2)
qt.disconnect(qobject1, signature1, qobject2, signature2)
