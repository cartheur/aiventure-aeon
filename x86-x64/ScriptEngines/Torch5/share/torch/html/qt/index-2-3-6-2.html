qstringlist:totable()

Torch Manual  >   QtLua and Qt Package Reference Manual  >   Package qt  >   Classes  >   qt.QStringList  >   qstringlist:totable()

Expression qstringlist:totable() returns a Lua table containing the elements of qstringlist starting at position 1.

Example 

 require 'qt'
 a = qt.QStringList{"a","b","c"}
 return a:totable()[2]
