qtdata:tobool()

Torch Manual  >   QtLua and Qt Package Reference Manual  >   Package qt  >   Qt Variants  >   qtdata:tobool()

Function qtdata:tobool() returns a boolean value associated with the value contained in the Qt variant. This is achieved by calling the Qt function QVariant::isNull. In the case of object pointers, this function can be used to determine if the object has been deleted or is currently allocated.
