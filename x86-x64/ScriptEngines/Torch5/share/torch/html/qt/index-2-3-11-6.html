[QLuaConsole signal] ttyEndOfFile()

Torch Manual  >   QtLua and Qt Package Reference Manual  >   Package qt  >   Classes  >   qt.qConsole  >   [QLuaConsole signal] ttyEndOfFile()

Signal ttyEndOfFile() is emitted whenever an end-of-file condition is detected on the console input. When this happens, qlua asks the user if he really wants to quit the application.
