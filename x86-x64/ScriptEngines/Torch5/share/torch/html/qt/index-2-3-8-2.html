qvariantmap:totable()

Torch Manual  >   QtLua and Qt Package Reference Manual  >   Package qt  >   Classes  >   qt.QVariantMap  >   qvariantmap:totable()

Expression qvariantmap:totable() returns a Lua table containing the contents of the dictonary qvariantmap.

Example 

 require 'qt'
 a = qt.QVariantMap{a=23,b=45,c="foo"}
 t = a:totable()
 return t.c
