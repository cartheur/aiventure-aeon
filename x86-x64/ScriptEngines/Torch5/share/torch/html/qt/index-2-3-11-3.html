qt.qConsole:addToHistory(string)

Torch Manual  >   QtLua and Qt Package Reference Manual  >   Package qt  >   Classes  >   qt.qConsole  >   qt.qConsole:addToHistory(string)

Function call qt.qConsole:addToHistory(string) adds string to the history of recent console input. This works only when qlua is compiled with command line history support.
