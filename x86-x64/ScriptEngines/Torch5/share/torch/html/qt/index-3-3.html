The luaQ functions

Torch Manual  >   QtLua and Qt Package Reference Manual  >   QtLua API  >   The luaQ functions

The luaQ functions complete the Lua API with facilities related to the QtLua interface. Some luaQ functions are only meaningful for C++ program. The following snipped ensure that all available functions are accessible in both C and C++. 

 #include "qtluautils.h"
 #ifdef __cplusplus
 # include "qtluaengine.h"
 #endif
