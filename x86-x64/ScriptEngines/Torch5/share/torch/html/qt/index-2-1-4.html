qtdata:type()

Torch Manual  >   QtLua and Qt Package Reference Manual  >   Package qt  >   Qt Variants  >   qtdata:type()

Function qtdata:type() returns a string naming the type of the value contained in the Qt variant. In the case of object pointers, this function returns a string of the form "ClassName*" where ClassName represents the object class. If the object has been deleted, this function returns nil.
