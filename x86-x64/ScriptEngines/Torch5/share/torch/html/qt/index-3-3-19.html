QVariant luaQ_toqvariant(lua_State *L, int i, int type = 0)

Converts the Lua value at position i of the stack into a Qt variant of type type. When type is zero, the most appropriate Qt variant type is returned. When the conversion is not possible, the function returns a Qt variant of type QVariant::Invalid.
