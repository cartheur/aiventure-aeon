[QLuaConsole signal] ttyBreak()

Torch Manual  >   QtLua and Qt Package Reference Manual  >   Package qt  >   Classes  >   qt.qConsole  >   [QLuaConsole signal] ttyBreak()

Signal ttyBreak() is emitted whenever the interrupt key sequence, typically Ctrl+C or Ctrl+Break is typed on the input console. When this happens, qlua stops the execution of Lua programs and waits for new commands on the console.
