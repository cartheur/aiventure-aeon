QObject* luaQ_toqobject(lua_State *L, int i, const QMetaObject *m = 0)

Returns a pointer to the Qt object located at position i on the stack. It returns a null pointer if the specified position does not represent a Qt object or if its class does not inherit the class represented by the meta object m. When m is null, all Qt objects are accepted.
