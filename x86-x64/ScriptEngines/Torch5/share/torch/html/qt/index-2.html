Package qt

Torch Manual  >   QtLua and Qt Package Reference Manual  >   Package qt

When you run program qlua, the package qt is readily accessible using require('qt') because the libqtlua library installs a suitable loading function in the array package.preload['qt'].

Package qt essentially contains the metatables associated with Lua userdata values and a few useful functions. The QtLua API also provides a means to name an arbitrary Qt object and publish it inside the package qt.
