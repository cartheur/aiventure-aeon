Classes qt.QSize and qt.QSizeF represent common Qt classes. Please refer to the Qt documentation for classes QSize and QSizeF for details.

Example: 

  require 'qtwidget'
  widget = qt.QWidget()
  widget.size = qt.QSize{width=320,height=200}
  widget:show()
