qt.QVariantList(table)

Torch Manual  >   QtLua and Qt Package Reference Manual  >   Package qt  >   Classes  >   qt.QVariantList  >   qt.QVariantList(table)

Expression qt.QVariantList(table) creates a new QVariantList initialized with the strings contained in table table at positions 1 to #table. When the argument table is omitted, an empty string list is returned.

Example 

 require 'qt'
 a = qt.QVariantList{"a",32}

 
qvariantlist:totable()
